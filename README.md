# FileSorter - 智能文件分类器

一个基于Python和PyQt5开发的桌面文件分类工具，具有苹果风格的毛玻璃界面，支持拖拽文件自动分类。

## ✨ 功能特性

- 🎨 **美观界面** - 苹果风格的毛玻璃效果UI
- 🖱️ **拖拽操作** - 简单拖拽即可完成文件分类
- 📁 **智能分类** - 根据文件扩展名自动分类
- 🔧 **高度可配置** - 支持自定义文件类型和分类规则
- 🖥️ **系统托盘** - 最小化到系统托盘，不占用桌面空间
- 🎯 **置顶显示** - 可设置窗口始终置顶

## 🚀 快速开始

### 环境要求
- Python 3.7+
- PyQt5

### 安装依赖
```bash
pip install -r requirements.txt
```

### 运行程序
```bash
python main.py
```

## 📖 使用说明

1. **首次运行**：程序会自动创建配置文件
2. **拖拽文件**：将文件拖拽到窗口中进行分类
3. **自定义设置**：右键系统托盘图标 → 设置，可修改文件分类规则
4. **最小化**：点击关闭按钮最小化到系统托盘

## ⚙️ 配置文件

程序使用 `config.json` 文件存储配置：

- `file_types`: 文件类型与目标文件夹的映射
- `window_settings`: 窗口位置、大小、透明度等设置

## 🛠️ 开发
