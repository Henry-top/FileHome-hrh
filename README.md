# fileHome - 智能文件管家

一个基于Python和PyQt5开发的桌面文件管理工具，具有苹果风格的毛玻璃界面，为你的文件提供一个温馨的"家"。

## 🏠 项目理念

fileHome 致力于让文件管理变得简单、优雅。就像为文件找到一个舒适的家一样，我们通过智能分类和美观界面，让你的数字生活更加整洁有序。

## ✨ 核心特性

- 🎨 **优雅界面** - 苹果风格的毛玻璃效果，视觉体验舒适
- 🏡 **智能归位** - 自动识别文件类型，为每个文件找到合适的"家"
- 🖱️ **轻松操作** - 简单拖拽即可完成文件整理
- 🔧 **个性定制** - 完全可配置的文件分类规则
- 🖥️ **贴心设计** - 系统托盘驻留，不打扰你的工作
- 🎯 **专注助手** - 置顶显示，随时待命

## 🚀 快速上手

### 环境要求
- Python 3.7+
- PyQt5

### 安装依赖
```bash
pip install -r requirements.txt
```

### 启动应用
```bash
python main.py
```

## 📖 使用指南

### 首次使用
1. 运行程序，fileHome会自动创建默认配置文件
2. 程序窗口会出现在桌面角落，随时准备为你服务

### 整理文件
1. **拖拽文件**：直接将文件拖到fileHome窗口中
2. **自动分类**：fileHome会根据文件类型自动归类
3. **完成整理**：文件会被移动到对应的文件夹中

### 个性化设置
1. **右键托盘图标** → **设置**
2. 修改文件类型与目标文件夹的对应关系
3. 保存设置，立即生效

## ⚙️ 配置说明

fileHome使用 `config.json` 文件存储你的个性化设置：

- `file_types`: 定义各种文件类型的"家"（目标文件夹）
- `window_settings`: 控制fileHome的外观和行为
